# HireLane

HireLane is a Career Command Center designed to help students and job seekers manage their job search journey in one place.

## MVP Features

### Authentication

* User Sign Up
* User Login
* User Logout

### Dashboard

* Stats Overview
* Upcoming Interviews
* Upcoming Deadlines
* Activity Timeline
* Quick Actions
* Application Pipeline Overview

### Applications

* Application Tracking
* Kanban Board
* Table View
* Add/Edit/Delete Applications
* Search & Filters
* Application Details Drawer
* Resume Tracking

### Interviews

* Interview Tracking
* Interview Rounds
* Preparation Notes
* Questions Asked
* Feedback
* Outcomes
* Interview Timeline

### Profile

* Personal Information
* Academic Information
* Career Preferences
* Resume Library
* Resume Management

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* Supabase
* PostgreSQL

### Deployment

* Vercel
